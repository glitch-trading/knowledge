# Python for Data Analysis — Wes McKinney

**The pandas creator's definitive guide to data manipulation in Python.**

## Why Read This

This is the foundational reference for working with data in Python. Written by the creator of pandas, it covers the entire data analysis stack: numpy for numerical computation, pandas for data manipulation, and matplotlib for visualization. If you're building trading systems, backtests, or data pipelines in Python, you will use these libraries daily.

Not a trading book — a tools book. But you can't do quantitative work without mastering these tools first.

## Key Takeaways

- **NumPy**: Vectorized operations, broadcasting, array indexing. The foundation everything else is built on. Avoid Python loops — use numpy operations for 100x speedups.
- **pandas**: DataFrames, Series, indexing (loc/iloc), groupby, merge/join, time series functionality, rolling windows. This is where you'll spend most of your time.
- **Data cleaning**: Handling missing data, type conversions, string operations, reshaping (pivot, melt, stack/unstack). Real data is always messy.
- **Time series**: DatetimeIndex, resampling, timezone handling, period arithmetic. Essential for financial data.
- **matplotlib**: Basic plotting. Sufficient for exploratory analysis; you'll likely graduate to plotly or seaborn for production visuals.
